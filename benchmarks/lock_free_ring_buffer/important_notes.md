### RT Throttling (Real-Time Throttling)

Linux kernel mechanism that prevents high-priority real-time tasks from monopolizing the CPU, which would otherwise cause system hangs. It typically limits RT tasks to using 95% of CPU time over a 1-second period.

### IRQ Affinity

IRQ affinity in Linux dictates which CPU core handles hardware interrupts (IRQs) from a specific device. By pinning specific interrupts to specific cores.
